# Day 10: SSH – Remote Command and Control

## Task 1 – Install OpenSSH
`pkg install openssh -y`

## Task 2 – Start the SSH Server
`sshd`

## Task 3 – Find Your Username
`whoami`

## Task 4 – Set a Password
`passwd`

## Task 5 – Find Your IP Address
`ifconfig` (look for 192.168.x.x)

## Task 6 – Connect from Another Device or Localhost
`ssh USERNAME@localhost -p 8022`

## Screenshot Required
Show successful SSH login (e.g., `ssh localhost -p 8022`)

## Absolute Truth
Remote access turns your phone into a server reachable from anywhere.
