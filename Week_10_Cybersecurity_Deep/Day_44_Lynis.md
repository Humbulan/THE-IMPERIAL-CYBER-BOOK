# Day 44: Lynis – Security Auditing Tool

## Objective
Run a comprehensive security scan on your VPS.

## Task 1 – Install Lynis
`sudo apt install lynis -y`

## Task 2 – Run an audit
`sudo lynis audit system`

## Task 3 – Review the suggestions (highlighted in red/green).

## Screenshot Required
Show a portion of the Lynis report (the summary or a few warnings).

## Absolute Truth
Automated security scanners catch what you might miss.
