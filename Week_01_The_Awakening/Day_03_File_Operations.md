# Day 3: Reading and Writing Files

## Commands to Learn
- `cat file` – read a file
- `echo "text" > file` – write to file (overwrites)
- `echo "more" >> file` – append to file
- `cp source dest` – copy a file
- `rm file` – delete a file

## Task 1 – Write Your Manifest

`echo "Goal: Build my stack" >> manifest.txt`
